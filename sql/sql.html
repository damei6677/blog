The LIKE form of CREATE TABLE allows you to copy an existing table definition exactly (without copying its data). In contrast to CTAS, the statement below creates a new empty_key_value_store table whose definition exactly matches the existing key_value_store in all particulars other than table name. The new table contains no rows.大致意思就是：CREATE TABLE创建表的形式允许您精确地复制现有表定义(不复制其数据)，其创建的表除了表名和源表不一样外，其余所有的细节都是一样的。但是没有源表的数据。所以这种create table like的形式非常适合对源表模式的复制。实际开发中其实用的也不是太多。
